# WebScraperPro

Zaawansowany web scraper z funkcjami eksportu i analizy danych.  
Stworzony w Pythonie z wykorzystaniem `aiohttp`, `BeautifulSoup` i `customtkinter`.

## Funkcje:
- Scrapowanie danych z wielu URL-i jednocześnie.
- Eksport do CSV, ZIP (wersja darmowa).
- Eksport do PDF (dostępny w wersji premium).
